
# 🌦️ Weather App

A weather application built with **C#** that provides real-time weather information through API integration. The application allows users to search for locations and view current weather conditions in a clean and user-friendly interface.

🌐 **Live Demo:**
https://safawain4-rgb.github.io/WeatherApp/

---

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ View current temperature
* ☁️ Display weather conditions
* 💧 Check humidity levels
* 🌬️ View wind speed information
* 📍 Location-based weather results
* 🔄 Real-time weather updates through API integration
* 📱 Responsive and user-friendly interface

## 🚀 Getting Started

### Prerequisites

* .NET SDK
* Visual Studio 2022 or later
* Internet connection for API requests

### Installation

1. Clone the repository

```bash
git clone https://github.com/SafaWain4-RGB/WeatherApp.git
```

2. Navigate to the project directory

```bash
cd WeatherApp
```

3. Restore dependencies

```bash
dotnet restore
```

4. Run the application

```bash
dotnet run
```
